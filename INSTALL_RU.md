# Простая установка Auto Support Generator

Эта инструкция рассчитана на пользователя, который не знаком с Python и GitHub.

## Что всё равно нужно установить отдельно

Auto Support Generator автоматизирует сборку SI, но не заменяет внешние
программы. Для полного workflow на компьютере должны быть установлены:

- Microsoft Word desktop;
- ChemDraw или ChemOffice с поддержкой OLE-объектов;
- MestReNova;
- Windows 10/11.

Python и Python-пакеты можно поставить через установочный `.bat` из этого
проекта.

Проверенные версии:

- ChemDraw `22.2.0.3300`;
- MestReNova `14.2.0-26256`;
- Python `3.12` или новее. Рекомендуемый вариант для установки: Python `3.12`.

## Как скачать

1. Откройте страницу проекта на GitHub.
2. Нажмите зелёную кнопку `Code`.
3. Выберите `Download ZIP`.
4. Распакуйте архив в удобную папку, например на рабочий стол.

## Как установить

Откройте распакованную папку и дважды кликните:

```text
Setup Auto SI Generator.bat
```

Скрипт сделает следующее:

1. найдёт установленный Python;
2. если Python не найден, попробует установить Python 3.12 через `winget`;
3. создаст локальную папку `.venv`;
4. установит все Python-пакеты;
5. проверит, что GUI импортируется.

Если `winget` недоступен, установите Python вручную:

```text
https://www.python.org/downloads/windows/
```

При установке Python включите галочку:

```text
Add python.exe to PATH
```

После этого снова запустите `Setup Auto SI Generator.bat`.

## Как запустить программу

После установки дважды кликните:

```text
Run Auto SI Generator.bat
```

Откроется графический интерфейс программы.

Старый файл `SI Generator GUI.bat` тоже работает: он просто запускает
`Run Auto SI Generator.bat`.

## Что делать, если установка не прошла

Чаще всего проблема связана с Python или RDKit.

Рекомендуемый вариант:

1. Установить Python 3.12 с python.org.
2. Перезапустить компьютер или PowerShell.
3. Запустить `Setup Auto SI Generator.bat` ещё раз.

Если проблема связана с Word, ChemDraw или MestReNova, проверьте, что эти
программы открываются вручную и активированы.
