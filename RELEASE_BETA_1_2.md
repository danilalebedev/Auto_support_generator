# Auto Support Generator beta 1.2

This beta release expands Auto Support Generator from the original SI generation workflow into a complete desktop workspace for creating and maintaining Supporting Information.

## Highlights

- New Fluent-style GUI with light and dark themes and built-in instructions.
- Separate Generate, Processing, Check, Patch and Add workflows.
- Per-run output folders with DOCX, input copies, spectra, Mnova files, reports and logs.
- Raw spectra can be supplied as a zip archive or a normal folder.
- Separate 1H and 13C MestReNova `.mngp` display profiles, thresholds and ppm ranges.
- Configurable peak picking, signal height and Bernstein/Whittaker baseline correction.
- PNG, clickable single-spectrum Mnova OLE, or no spectra appendix.
- Reaction loading and yield calculations from Reaction schema and Scope Word files.
- NMR, HRMS and elemental-analysis checks with structured reports.
- Add compounds to the same series or with a new synthetic method.
- Patch an existing SI by renumbering, removing, reordering or swapping compounds without reprocessing spectra.
- Three complete editable example datasets and Russian/English user guides.

## Installation

Download `AutoSupportGeneratorSetup.exe` from the GitHub release assets and run it on Windows. Microsoft Word, ChemDraw and MestReNova must be installed separately.

This is a research beta. Keep copies of original input data and review generated SI before publication.
